# Proveedores Internacionales

## Descripcion
Script de sql para extraccion y analisis de datos en el proyecto ARIS.

## Tipo
Base de datos: Oracle
Categoria: Terceros

## Codigo SQL

```sql
SELECT  
    J.NUMERO_DOCUMENTO, 
    J.TIPDOC_CODIGO, 
    J.NOMBRE_DEL_CLIENTE,
    J.DIVPOL_CODIGO_REGISTRADO_EN,
    DP1.NOMBRE AS DIVPOL_NOMBRE_REGISTRADO_EN,    
    DP1.NACIONALIDAD AS DIVPOL_NACIONALIDAD_REGISTRADO_EN,    
    J.DIVPOL_CODIGO_NACIDO_EN,
    DP2.NOMBRE AS DIVPOL_NOMBRE_NACIDO_EN,
    DP2.NACIONALIDAD AS DIVPOL_NACIONALIDAD_NACIDO_EN
FROM 
    (
        SELECT 
            NUMERO_DOCUMENTO, 
            TIPDOC_CODIGO, 
            RAZON_SOCIAL AS NOMBRE_DEL_CLIENTE,
            DIVPOL_CODIGO AS DIVPOL_CODIGO_REGISTRADO_EN,
            DIVPOL_CODIGO_NACIDO_EN
        FROM JURIDICOS

        UNION ALL

        SELECT 
            NUMERO_DOCUMENTO, 
            TIPDOC_CODIGO, 
            PRIMER_NOMBRE || ' ' || SEGUNDO_NOMBRE || ' ' || PRIMER_APELLIDO || ' ' || SEGUNDO_APELLIDO AS NOMBRE_DEL_CLIENTE,
            DIVPOL_CODIGO_REGISTRADO_EN,
            DIVPOL_CODIGO_NACIDO_EN
        FROM NATURALES
    )  J
LEFT JOIN 
    DIVISION_POLITICAS DP1 ON J.DIVPOL_CODIGO_REGISTRADO_EN = DP1.CODIGO
LEFT JOIN 
    DIVISION_POLITICAS DP2 ON J.DIVPOL_CODIGO_NACIDO_EN = DP2.CODIGO
WHERE (
        J.TIPDOC_CODIGO = 'NE' OR 
        DP2.NACIONALIDAD != 'COLOMBIANO' OR 
        DP1.NACIONALIDAD != 'COLOMBIANO')

```
